# 3D Interactive Web Portfolio

A modern, professional portfolio website featuring responsive, real-time 3D elements, cutting-edge design aesthetics, and interactive features.

## 🌟 Features

- **Interactive 3D Geometry**: Dynamic Three.js geometric shapes that respond intelligently to mouse and cursor movements.
- **Terminal Mode**: A sleek, text-based interactive terminal view simulating an OS-like developer environment.
- **Adaptive Theme**: One-click toggle between elegant Dark and Light modes.
- **Responsive Design**: Flawlessly adapts to all screen sizes, from mobile devices to ultrawide monitors.
- **Smooth Animations**: Subtle micro-animations, scroll-based section fade-ins, and soft interactive states.

## 🛠️ Technology Stack

- **Core**: HTML5, Vanilla CSS3, JavaScript (ES6+)
- **3D Graphics**: [Three.js](https://threejs.org/)
- **Icons**: Devicons, Feather Icons (SVG)
- **Deployment & Architecture**: Pure client-side delivery, no build steps required for the core visual framework.

## 🚀 Getting Started

Simply clone the repository and open `index.html` in any modern web browser to view the portfolio locally. No `npm` installation required!

```bash
git clone https://github.com/Shashikant-codes/3d_portfolio.git
cd 3d_portfolio
```
